# 🌦️ Weather App (React + Open-Meteo API)

A modern weather application built using React and Chart.js that shows real-time weather and a 24-hour forecast with graphs.

## 🚀 Live Demo  
https://weather-app-mu-ruby-19.vercel.app/

## ✨ Features
- 🌍 Auto-detect user location (Geolocation API)
- 🔍 Search weather by city
- ⏰ Next 24 hours forecast
- 📈 Temperature trend graph (Chart.js)
- 🌧️ Rain probability graph
- 🎨 Dynamic background based on weather
- 💾 Remembers last searched city (localStorage)
- 📱 Fully responsive UI

## 🛠️ Tech Stack
- React (Vite)
- Chart.js + react-chartjs-2
- Open-Meteo API
- HTML, CSS, JavaScript
- GitHub + Vercel
